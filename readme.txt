== Description ==

When users access the website, the system will automatically save the first url to the COOKIE (applicable to LEADHUD software).

== Installation ==

1. Upload the plugin cookies-for-leadhub files to the `/wp-content/plugins/` directory

2. Activate the cookies-for-leadhub plugin through the 'Plugins' menu in WordPress.
